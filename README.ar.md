## نبذة عني

أعمل كقائد ومسؤول لتقنية المعلومات في شركة تعمل ضمن بيئة EPC. لا يقتصر دوري على إدارة أجهزة المستخدمين أو معالجة الأعطال اليومية، بل يشمل تصميم البنية التحتية وتشغيل الشبكات والخوادم وتأمين الوصول وإدارة خدمات الويب والبريد والنسخ الاحتياطي والمراقبة واستمرارية الخدمات.

أتعامل مع المشكلة من طبقة الشبكة والأنظمة حتى الخدمة والتطبيق وتجربة المستخدم. هدفي هو بناء حلول مستقرة، قابلة للصيانة، موثقة ومناسبة للاحتياجات الفعلية للعمل.

إلى جانب البنية التحتية والشبكات، أعمل أيضًا في تصميم مواقع الويب، وإعداد العروض التقديمية التقنية والمؤسسية، والتوثيق، والتصميم البصري، وتحويل المعلومات المعقدة إلى محتوى واضح وقابل للعرض.

## مجالات الخبرة

### البنية التحتية وإدارة الأنظمة

- إدارة واستكشاف أعطال خوادم `Linux` و`Ubuntu Server`
- العمل مع `Windows Server` والخدمات المؤسسية
- إدارة البيئات الافتراضية والآلات الافتراضية وتخطيط الموارد
- تصميم النسخ الاحتياطي والاستعادة والتحقق من سلامة النسخ
- المراقبة، تحليل السجلات، إدارة الخدمات وتشخيص الأعطال
- تقوية الخوادم، إدارة الصلاحيات، SSH وأساسيات Hardening
- دعم المستخدمين والأنظمة والمعدات والخدمات الداخلية

### الشبكات وأمن الاتصالات

- إدارة متقدمة لـ `MikroTik RouterOS`
- Routing، NAT، Firewall، VLAN، Subnetting وPolicy-Based Routing
- تصميم Multi-WAN، Failover ومسارات خروج منفصلة
- تنفيذ `WireGuard` و`OpenVPN` و`L2TP`
- بناء وصول آمن عن بُعد وربط المستخدمين والخوادم والمواقع
- Port Forwarding، التحكم بالوصول، التصفية وتشخيص الاتصال
- تحليل المشكلات من مستوى Packet وRoute وFirewall حتى الخدمة النهائية

### الويب والنطاقات والبريد

- تشغيل وإدارة `Nginx` و`Caddy` و`HAProxy` و`HestiaCP`
- إدارة Domain وDNS وSSL/TLS والشهادات
- إعداد Web Server وReverse Proxy واستكشاف أعطالهما
- إدارة Mail Server وSMTP Relay ومسارات إرسال البريد ومشكلات التسليم
- تصميم وتطوير وصيانة مواقع الشركات باستخدام `WordPress`
- العمل باستخدام `HTML` و`CSS` و`JavaScript`
- نقل الخدمات والنطاقات والمواقع مع تقليل Downtime

### البرمجة والأتمتة

- استخدام `Python` لحل المشكلات وبناء الأدوات وأتمتة المهام
- استخدام `Bash` لإدارة وأتمتة أنظمة Linux
- العمل باستخدام `C#` وتطوير الألعاب والأدوات عبر `Unity`
- استخدام `Git` و`GitHub` لإدارة الإصدارات والتوثيق والمشاريع
- تصميم Scripts قابلة لإعادة الاستخدام لعمليات البنية التحتية
- تحويل الإجراءات اليدوية والمتكررة إلى Workflows موثوقة

### التصميم والتواصل التقني

- إعداد عروض تقديمية مؤسسية وإدارية وتقنية
- تصميم المخططات وFlowcharts والتمثيل البصري لهندسة الأنظمة
- التصميم والتحرير باستخدام `Photoshop`
- تصميم واجهات الويب وبنية المحتوى للمواقع المؤسسية
- كتابة التوثيق الفني وأدلة التنفيذ والتقارير
- تبسيط المعلومات التقنية للمديرين والمستخدمين والفرق غير التقنية

### العمليات المؤسسية

- إدارة معدات وخدمات وصلاحيات وأصول تقنية المعلومات
- دعم أنظمة الحضور والانصراف وVoIP والخدمات الداخلية
- تخطيط النسخ الاحتياطية الدورية ومراقبة استمرارية الخدمات
- التنسيق مع الأقسام المختلفة لحل الاحتياجات التقنية وتحسين Workflows
- تقييم الحلول من حيث التكلفة والمخاطر وسهولة الصيانة والاحتياج الحقيقي

## التدريب ومسار التعلم

أكملت برامج تدريبية في `Python` والذكاء الاصطناعي لدى **Irancell Labs**. كما تلقيت تدريبًا في `Network+` و`Security+` وLinux وWindows Server وMikroTik وWordPress وتصميم الويب وPhotoshop وC#.

أعمل حاليًا على تطوير مساري نحو `DevOps` و`Cloud Engineering` مع التركيز على:

- Docker وDocker Compose
- CI/CD وGitHub Actions
- Infrastructure as Code
- Ansible وTerraform
- Kubernetes وHelm
- Cloud Platforms
- Prometheus وGrafana وLogging
- Linux Automation وInfrastructure Security
