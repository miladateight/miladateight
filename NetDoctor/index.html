Windows Network Diagnostics

Find the network fault before touching the settings.

NetDoctor is a guided Windows tool for diagnosing DNS, gateway, proxy, latency, packet loss, VPN-like adapters, local access, international access, and safe repair options without burying the user in network jargon.

What it checks

Clear diagnostics for messy network problems.

NetDoctor starts from the user-visible problem and builds a plain-language report from the network layers that usually break first.

01

Adapter and gateway

Reads active adapter state, gateway availability, DNS configuration, and common route symptoms.

02

DNS and reachability

Tests DNS resolution, local access, international access, TCP port reachability, latency, and packet loss.

03

Proxy and VPN clues

Detects Windows proxy state and VPN-like adapters that can quietly change the user's path to the internet.

04

Human diagnosis

Turns raw checks into a summary that explains what probably failed and where to look next.

05

Safe repair

Repair actions save previous settings first and expose undo where a reversible fix is available.

06

Windows installer

The release ships as a Windows setup file so users can install the app without building from source.

Flow

Problem first, repair second.

step 1

Ask what is broken

The app starts from the symptom the user actually sees instead of dumping a technical checklist.

step 2

Run layered checks

It checks adapter, route, DNS, latency, ports, proxy, VPN-like adapters, and access paths.

step 3

Explain the result

The output is written for a normal user while still preserving useful signals for a technical operator.

step 4

Offer safe fixes

When a reversible fix exists, NetDoctor prepares it with a saved previous state and an undo path.
